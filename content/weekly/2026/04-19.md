## やったこと

- 個人用の家計アドバイスアプリを作った
  - claude code をもうちょい使いこなしたくて個人で作れる適当な題材ないかな〜と探していて、僕は家計管理がザルなのでアドバイスをもらえるアプリがあると良いなと思って作った。完全に個人用なので、PWA として作りつつベーシック認証をかけた上で公開している
  - 仕様
    - マネフォの月ごとの収支データ CSV をそのアプリにアップロードすると AI が家計へのアドバイスをしてくれる、というもの
      - 現在の借金情報を事前にコンテキストとして登録しておきつつ、毎月の CSV が溜まっていくと、借金とか月ごとの変化とかの観点でのアドバイスも出してくれるようになっている
    - また、購入判断機能もあって、ほしいものと値段と用途を入力すると、それを買っていいかどうかを教えてくれる
      - これもまたアプリ内に溜まった情報をコンテキストとして使った上でアドバイスしてくれる
  - 使ってみた感想
    - ある程度わかっていたことだけれど、当たり前のことしか言ってこない。コンテキストを与えているものの、お金にルーズな僕から見てもそれはそうでしょうねっていうことしか言わないので、ありがたみがあまりないかも
      - 購入判断に関しても、何を聞いても大体「条件付きでOK」っていう結論を返してくる
    - もう少しデータが蓄積していけば洗練されていく可能性はあるのでもうしばらく様子を見たい
  - 今後
    - 毎月マネフォから CSV を落としてこのアプリにつっこむっていうのがめんどくさいので、勝手にマネフォを見に行ってくれるようにしたい
    - あと毎月やっている面倒くさい作業として、マネフォの自動カテゴリ振り分けが間違っていることが多々あるので、それをレシートを見ながら修正する作業があって、レシートの写真をまとめて撮って渡したらマネフォを操作してカテゴリを正しくする作業とかもやってもらいたいな
  - 作ってみた感想
    - 最初は仕事でやるようにちゃんと設計書を作ったり、AI が書いたコードを全部レビューしたりしていたんだけど、それだとやっぱり自分がボトルネックになって大して速度が出ないので、最終的にはコードは一切見ないで作ったものをすぐに動かしてレビューする形になっていった
    - それをやる上で便利だったのが2つあった
      - claude code の remote control でデスクトップ PC で起動している claude のセッションにスマホからアクセスして、常にスマホから指示を出すことで作業を進められた
      - vercel にデプロイすることで、claude が修正したものを即座にプッシュして、プッシュをフックに vercel が最新をデプロイしてくれるので、先に書いた claude の remote control と合わせて、スマホで指示したものが即座にデプロイされてスマホから動作確認できた
    - というところでバイブコーディングの体験というのは本当に良かったんだけど、やっぱり僕はコードを一切見ずに作ったものを不特定多数が使うアプリとしては出すのは怖くてできないので、こういうのは完全に個人用のものを作るのにのみ使うべきだなと思った
      - そういう面で今回の題材はとても適していてちょうどよかった
      - まあセキュリティ面のレビュー体制も仕組みとしてちゃんと整えたら問題ないのかも
- Fresh and Blood というカードゲームを体験した
  - 会社の同僚が [Fresh and Blood](https://fabtcg.com/ja/) というカードゲームにハマっていて、おすすめされたので会社のメンバーで集まって遊ぶ会をした
  - ルールが複雑でめっちゃ難しくて、特に僕が触っていた「エニグマ」というヒーローは考えることが多くて、頭がいっぱいいっぱいになってしまった
  - とはいえめっちゃ面白かったので、またやりたい気持ちと今ここに時間をかけるべきか、という気持ちがせめぎ合っている
  - 人におすすめされたものをやってみるっていうのは体験としてとても良かった
- やったゲーム
  - [PRAGMATA](https://www.capcom-games.com/pragmata/ja-jp/)
    - まじで面白い。シューティングとパズルゲームのバランスが良くて、アクションしつつも頭を使う感じがとても良い
    - 相棒の子どもが尊すぎて、多分最後は悲しいことになるんだろうなっていう結末のことを考えてしんどくなってくる
